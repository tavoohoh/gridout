Getting Started:

Gridout is a 12 column layout that uses CSS grid technology.
Download the CSS stylesheet, add the appropriate classes to your markup, and start building beautiful UI.

Column system:

If you have ever used another responsive framework, the column system will seem familiar.

4 of 12
3 of 12
5 of 12
<div class="grid"> <div class="col-4">4 of 12</div> <div class="col-3">3 of 12</div> <div class="col-5">5 of 12</div> </div>

Responsive columns:

Available break points are:

  • col-#/col-sm-# Landscape phones, 576px and up.
  • col-md-# Medium devices,tablets, 768px and up.
  • col-lg-# Large devices, desktops, 992px and up.
  • col-xl-# Extra large devices, large desktops, 1200px and up.

Viewport size is "sm" all col-# or col-sm-# columns are being applied. Viewport size is "md" all col-md-# columns are being applied. Viewport size is "lg" all col-lg-# columns are being applied. Viewport size is "xl" all col-xl-# columns are being applied. To see other applications try changing the size of the window.

col-6 col-md-4 col-lg-4 col-xl-3
col-6 col-md-8 col-lg-4 col-xl-6
col-12 col-md-12 col-lg-4 col-xl-3
<div class="grid"> <div class="col-6 col-md-4 col-lg-4 col-xl-3"> sm: 6 / md: 4 / lg: 4 / xl: 3 of 12. </div> <div class="col-6 col-md-8 col-lg-4 col-xl-6"> sm: 6 / md: 8 / lg: 4 / xl: 6 of 12. </div> <div class="col-12 col-md-12 col-lg-4 col-xl-3"> sm: 12 / md: 12 / lg: 4 / xl: 3 of 12. </div> </div>

Custom column position:

This implementation can be a bit confusing if you are not used to CSS grid.

.col-custom | --ini: 2; --end: 8;
.col-custom | --ini: 4; --end: 11;
<div class="grid"> <div class="col-custom" style="--ini: 2; --end: 8;"> <span> Column start at grid position 2 And end at grid position 7 </span> </div> <div class="col-custom" style="--ini: 4; --end: 11;"> <span> Column start at grid position 4 And end at grid position 10 </span> </div> </div>
